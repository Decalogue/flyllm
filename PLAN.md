## 📝 复习重点总结

### 大模型八股核心（必须手撕）
1. **MHA**：标准实现、维度变换、复杂度
2. **GQA**：广播机制、参数量对比
3. **RoPE**：旋转矩阵、外推能力
4. **Flash Attention**：分块原理、内存优化
5. **KV Cache**：推理优化、内存管理

### Leetcode高频题型
1. **数组/字符串**：双指针、滑动窗口
2. **链表**：反转、合并、快慢指针
3. **二叉树**：遍历、递归、DFS/BFS
4. **动态规划**：背包、LIS、编辑距离
5. **回溯**：排列、组合、N皇后

### Agent+Memory核心架构
1. **HindSight**：Retain、Recall、Reflect
2. **CogMem**：FoA、DA、LTM三层记忆
3. **LightMem**：感觉记忆、短时记忆、长时记忆
4. **MemMachine**：情景记忆、语义记忆、用户画像
5. **A-Mem**：Zettelkasten方法、动态索引
6. **O-Mem**：主动用户画像、分层检索
