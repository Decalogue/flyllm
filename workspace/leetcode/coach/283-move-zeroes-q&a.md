## 💾 学习笔记沉淀

### 核心模板提炼

```python
# 快慢指针原地重排通用模板
def move_by_condition(nums, condition):
    """
    condition: 判断元素是否应该前移的函数
    """
    slow = 0
    for fast in range(len(nums)):
        if condition(nums[fast]):
            nums[slow], nums[fast] = nums[fast], nums[slow]
            slow += 1
    return nums
```

### 记忆锚点

- **核心痛点**：如何在O(1)额外空间内完成元素重排
- **通用模式**：快慢指针，slow负责"放"，fast负责"找"
- **关键细节**：先判断后交换，slow只有在有效元素时才移动
