# 621. 任务调度器

**标签**：`贪心` `中等`  
**难度**：⭐⭐ 中等  

---

## 思路

设最高频任务数 `maxc`，其个数 `nmax`；答案至少 `(maxc-1)*(n+1)+nmax`。

---

## 代码

```python
from collections import Counter


class Solution:
    def leastInterval(self, tasks: list[str], n: int) -> int:
        c = Counter(tasks)
        mx = max(c.values())
        nmax = sum(1 for v in c.values() if v == mx)
        return max(len(tasks), (mx - 1) * (n + 1) + nmax)
```
