# 226. 翻转二叉树

**标签**：`树` `DFS` `简单`  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 代码实现

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def invertTree(self, root: TreeNode | None) -> TreeNode | None:
        if not root:
            return None
        root.left, root.right = self.invertTree(root.right), self.invertTree(
            root.left
        )
        return root
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(h)
