## 📝 使用说明

### 如何使用本列表：

1. **按分类刷题**：建议先专注于同一分类的题目，掌握核心算法思想
2. **标记进度**：在本地克隆本文件，用 ✅ 标记完成、🔄标记复习中、❌标记暂时跳过
3. **配合 LeetCode**：每道题都可用题号在 LeetCode 官网快速搜索
4. **重点突破**：建议优先完成 Medium 难度题目，面试高频

### 建议刷题顺序：

```
数组/链表 → 二叉树 → 栈队列 → 回溯 → 动态规划 → 图论 → 高级数据结构
```

### 面试重点：

- **Easy**: 20, 70, 141, 206, 226 等基础题
- **Medium**: 1, 3, 15, 46, 78, 96, 102, 104, 121, 200, 207, 226 等高线
- **Hard**: 4, 76, 124, 146, 239, 297, 301 等拔高题
