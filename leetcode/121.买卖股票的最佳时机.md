# 121. 买卖股票的最佳时机

**标签**：`数组` `贪心` `简单`  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        mn = float("inf")
        ans = 0
        for p in prices:
            mn = min(mn, p)
            ans = max(ans, p - mn)
        return ans
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(1)
