# 50. Pow(x, n)

**标签**：`递归` `数学` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 中频

---

## 解题思路

**快速幂**：n 为偶数时 `x^n = (x^2)^(n/2)`；为奇数时 `x * x^(n-1)`。注意 n 为负数时取倒数。

---

## 代码实现

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            x = 1 / x
            n = -n
        res = 1.0
        while n:
            if n & 1:
                res *= x
            x *= x
            n >>= 1
        return res
```

---

## 复杂度

- **时间**：O(log|n|)，**空间**：O(1)
