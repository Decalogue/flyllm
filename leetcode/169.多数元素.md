# LeetCode 169. 多数元素

**题号**: 169
**难度**: Easy
**标签**: 数组, 哈希表, 分治, 摩尔投票法

---

## 🎯 题目描述

给定一个大小为 `n` 的数组，找到其中的**多数元素**。多数元素是指在数组中出现次数 **大于 ⌊ n/2 ⌋ 的元素**。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

**示例 1**:
```python
输入: nums = [3,2,3]
输出: 3
```

**示例 2**:
```python
输入: nums = [2,2,1,1,1,2,2]
输出: 2
```

---

## 💡 解题思路

### 方法1：摩尔投票法（最优解）

**核心思想**：利用"多数元素数量 > n/2"的性质，通过票数抵消找到候选人

**算法步骤**:
1. 初始化 `candidate = None`, `count = 0`
2. 遍历数组每个元素 `num`：
   - 如果 `count == 0`，设置 `candidate = num`, `count = 1`
   - 如果 `num == candidate`，`count += 1`
   - 否则 `count -= 1`
3. 返回 `candidate`（题目保证存在多数元素）

**正确性证明**:
假设多数元素是 `x`，出现 `m` 次，且 `m > n - m`。
最坏情况下，每个 `x` 都被一个其他元素抵消，最终票数差为 `m - (n - m) = 2m - n > 0`，所以 `x` 必然存活到最后。

### 方法2：哈希表统计

**算法步骤**:
1. 使用哈希表统计每个元素出现次数
2. 找到出现次数最多的元素
3. 验证是否大于 n/2

**复杂度**:
- 时间复杂度: O(n)
- 空间复杂度: O(n) ❌ 不符合要求

### 方法3：排序

**算法思路**:
由于多数元素出现次数 > n/2，排序后中间位置的元素一定是多数元素。

```python
def majority_element(nums):
    nums.sort()
    return nums[len(nums) // 2]
```

**复杂度**:
- 时间复杂度: O(n log n) ❌ 不够优
- 空间复杂度: O(1) 或 O(n)（取决于排序算法）

## 📝 Python 实现

### 实现1：摩尔投票法（最优）

```python
from typing import List

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        """
        摩尔投票法 - Boyer-Moore Majority Vote Algorithm
        时间复杂度: O(n)
        空间复杂度: O(1)
        """
        candidate = None
        count = 0
        
        for num in nums:
            if count == 0:
                candidate = num
                count = 1
            elif num == candidate:
                count += 1
            else:
                count -= 1
        
        return candidate
```

### 实现2：带验证的完整版本

如果题目不保证多数元素一定存在：

```python
from typing import List

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        """
        摩尔投票法 + 验证
        """
        # 第一阶段：找出候选人
        candidate = None
        count = 0
        
        for num in nums:
            if count == 0:
                candidate = num
                count = 1
            elif num == candidate:
                count += 1
            else:
                count -= 1
        
        # 第二阶段：验证候选人
        count = sum(1 for num in nums if num == candidate)
        
        if count > len(nums) // 2:
            return candidate
        else:
            return None  # 不存在多数元素
```

## 🔍 边界情况

### 1. 单个元素
```python
nums = [1]
# candidate=1, count=1
# 返回 1
```

### 2. 两个相同元素
```python
nums = [2, 2]
# 第一个2: candidate=2, count=1
# 第二个2: candidate=2, count=2
# 返回 2
```

### 3. 大量不同元素
```python
nums = [1, 2, 3, 4, 5, 5, 5, 5, 5, 5]
# 5出现了6次 > 10/2=5
# 最终 candidate = 5
```

## 💡 记忆锚点

### 摩尔投票算法模板

```python
def majority_element(nums):
    """
    Boyer-Moore Majority Vote Algorithm
    适用于: 找出出现次数 > n/2 的众数
    """
    candidate = None
    count = 0
    
    for num in nums:
        if count == 0:
            candidate = num
            count = 1
        elif num == candidate:
            count += 1
        else:
            count -= 1
    
    # 题目保证存在多数元素
    return candidate
```

### 关键要点
1. **核心思想**：票数抵消，多数元素必然存活
2. **时间空间**：O(n)时间，O(1)空间
3. **适用条件**：必须保证多数元素存在
4. **验证机制**：如果不保证存在，需要第二遍遍历验证
