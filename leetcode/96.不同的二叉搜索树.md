# 96. 不同的二叉搜索树

**标签**：`动态规划` `卡特兰数` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def numTrees(self, n: int) -> int:
        dp = [0] * (n + 1)
        dp[0] = dp[1] = 1
        for i in range(2, n + 1):
            for j in range(1, i + 1):
                dp[i] += dp[j - 1] * dp[i - j]
        return dp[n]
```

---

## 复杂度

- **时间**：O(n²)，**空间**：O(n)
