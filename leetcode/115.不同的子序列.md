# 115. 不同的子序列

**标签**：`动态规划` `字符串` `困难`  
**难度**：⭐⭐⭐ 困难  

---

## 代码

```python
class Solution:
    def numDistinct(self, s: str, t: str) -> int:
        m, n = len(s), len(t)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(m + 1):
            dp[i][0] = 1
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                dp[i][j] = dp[i - 1][j]
                if s[i - 1] == t[j - 1]:
                    dp[i][j] += dp[i - 1][j - 1]
        return dp[m][n]
```

---

## 复杂度

- O(mn) 时间，O(mn) 空间（可滚为一维）
