# 142. 环形链表 II

**标签**：`链表` `数学` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 解题思路

快慢相遇后，一指针回 head，两指针同速再走，相遇点即环入口。

---

## 代码实现

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def detectCycle(self, head: ListNode | None) -> ListNode | None:
        slow = fast = head
        while fast and fast.next:
            slow, fast = slow.next, fast.next.next
            if slow is fast:
                p = head
                while p is not slow:
                    p, slow = p.next, slow.next
                return p
        return None
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(1)
