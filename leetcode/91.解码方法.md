# 91. 解码方法

**标签**：`动态规划` `字符串` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def numDecodings(self, s: str) -> int:
        if not s or s[0] == "0":
            return 0
        n = len(s)
        dp = [0] * (n + 1)
        dp[0] = dp[1] = 1
        for i in range(2, n + 1):
            if s[i - 1] != "0":
                dp[i] += dp[i - 1]
            two = int(s[i - 2 : i])
            if 10 <= two <= 26:
                dp[i] += dp[i - 2]
        return dp[n]
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(n) 可优化为 O(1)
