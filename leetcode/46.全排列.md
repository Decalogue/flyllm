# 46. 全排列

**标签**：`回溯` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def permute(self, nums: list[int]) -> list[list[int]]:
        ans: list[list[int]] = []

        def dfs(path: list[int], used: list[bool]) -> None:
            if len(path) == len(nums):
                ans.append(path.copy())
                return
            for i in range(len(nums)):
                if used[i]:
                    continue
                used[i] = True
                path.append(nums[i])
                dfs(path, used)
                path.pop()
                used[i] = False

        dfs([], [False] * len(nums))
        return ans
```

---

## 复杂度

- **时间**：O(n·n!)，**空间**：O(n)
