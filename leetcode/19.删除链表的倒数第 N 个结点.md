# 19. 删除链表的倒数第 N 个结点

**标签**：`链表` `双指针` `中等`  
**分类**：链表、快慢指针  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给你一个链表，删除链表的**倒数第 `n` 个结点**，并返回链表的头结点。

### 示例 1

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
```

### 示例 2

```
输入：head = [1], n = 1
输出：[]
```

### 示例 3

```
输入：head = [1,2], n = 1
输出：[1]
```

### 提示

- 链表中结点数目为 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：用一个**虚拟头节点** `dummy`，让快指针先走 `n+1` 步，再快慢同速前进；当快指针到 `None` 时，慢指针正好停在待删节点的前一个，执行 `slow.next = slow.next.next`。

**关键词**：`快慢指针` `固定间距 n` `虚拟头节点`

### 复杂度速记

- **时间复杂度**：O(L)，L 为链表长度。  
- **空间复杂度**：O(1)。

### 记忆口诀

「快先走 n+1，慢再一起走；快到尾，慢在删前口。」

---

## 解题思路

### 为什么要虚拟头节点

如果要删除的是原头结点（例如 `[1], n=1` 或 `[1,2], n=2`），直接操作会有分支判断。  
加上 `dummy -> head` 后，删除任意节点都统一成「删除 `slow.next`」，代码更稳定。

### 双指针不变量

1. 初始化：`fast = slow = dummy`。  
2. 让 `fast` 先走 `n+1` 步，此时 `fast` 与 `slow` 相距 `n+1`。  
3. 再让两者同步前进，直到 `fast` 走到 `None`。  
4. 由于间距恒定，`slow` 恰好位于倒数第 `n` 个节点的前一个。  
5. 删除：`slow.next = slow.next.next`。

---

## 代码实现

### 写法一：一次遍历快慢指针（推荐）

```python
from typing import Optional


class ListNode:
    def __init__(self, val: int = 0, next: Optional["ListNode"] = None):
        self.val = val
        self.next = next


class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = slow = dummy

        # fast 先走 n+1 步，确保 slow 最终停在待删节点前一位
        for _ in range(n + 1):
            fast = fast.next

        while fast:
            fast = fast.next
            slow = slow.next

        slow.next = slow.next.next
        return dummy.next
```

### 写法二：两次遍历（长度法，便于理解）

先统计链表长度 `L`，要删除的是正数第 `L - n + 1` 个节点；再从 `dummy` 走到其前驱删除。

```python
from typing import Optional


class ListNode:
    def __init__(self, val: int = 0, next: Optional["ListNode"] = None):
        self.val = val
        self.next = next


class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        cur = head
        length = 0
        while cur:
            length += 1
            cur = cur.next

        steps = length - n
        cur = dummy
        for _ in range(steps):
            cur = cur.next

        cur.next = cur.next.next
        return dummy.next
```

---

## ⚠️ 常见陷阱

1. **快指针先走步数写错**  
   - 若 `fast` 从 `dummy` 出发，应先走 **`n+1`** 步，不是 `n` 步。

2. **不加虚拟头导致删头结点分支混乱**  
   - 用 `dummy` 可以统一处理所有删除位置。

3. **删除语句写成 `slow = slow.next.next`**  
   - 正确是改链：`slow.next = slow.next.next`，不是移动指针变量。

4. **以为必须用两次遍历**  
   - 一次遍历快慢指针就能做，且是面试标准解。

## 总结

19 的关键是「**固定快慢指针间距** + **虚拟头节点**」。让快指针先走 `n+1`，再同步前进，慢指针落在待删节点前驱，最后改一次 `next` 即可完成删除。
