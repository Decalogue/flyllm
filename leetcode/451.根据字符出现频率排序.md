# 451. 根据字符出现频率排序

**标签**：`哈希表` `字符串` `桶排序` `排序` `中等`  
**分类**：字符串、频数统计  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 中高频

---

## 题目描述

给定字符串 `s`，按字符出现**频率从高到低**重新排列，返回新的字符串。若多个字符出现次数相同，它们之间的**相对顺序任意**。

### 示例 1

```
输入：s = "tree"
输出："eert" 或 "eetr"
解释：'e' 出现 2 次，'r'、't' 各 1 次。
```

### 示例 2

```
输入：s = "cccaaa"
输出："cccaaa" 或 "aaaccc"
```

### 示例 3

```
输入：s = "Aabb"
输出："bbAa" 或 "bbaA"
说明：区分大小写，'A' 与 'a' 视为不同字符。
```

### 提示

- `1 <= s.length <= 5 * 10^5`
- `s` 由**大小写字母和数字**组成

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：先**统计频数**，再按频数**降序**把每个字符重复拼接成串。

**关键词**：`Counter` `按值排序` `桶排序`

### 复杂度速记

- **哈希 + 排序字符种类**：O(n + k log k)，k 为不同字符数（≤ |Σ|）。  
- **桶排序频数**：O(n) 时间、O(n) 空间（与题解实现相关）。

### 记忆口诀

「先数数，再按次数从大到小拼回去。」

---

## 解题思路

### 核心思想

1. 遍历 `s` 得到每个字符出现次数（哈希表 / `Counter`）。  
2. 将 `(字符, 次数)` 按**次数从大到小**排序。  
3. 对每个字符输出 `次数` 个该字符，拼成答案。

### 写法对比

| 方法 | 做法 | 适用 |
|------|------|------|
| 排序 | `sorted(items, key=-count)` | 实现最短，k 不大时足够 |
| 堆 | 按频数用大根堆弹出 | k 大、强调堆时 |
| 桶 | 下标=频数，桶里放字符列表 | 追求 O(n) 且字符集有限 |

本题 `s` 最长 5×10⁵，不同字符 k 最多约 62（数字+大小写）或扩展 Unicode，**排序 k** 通常远小于 O(n log n) 的 n。

### 算法步骤（排序法）

1. `cnt = Counter(s)`。  
2. `pairs = sorted(cnt.items(), key=lambda x: -x[1])`。  
3. `return "".join(ch * c for ch, c in pairs)`。

---

## 代码实现

### 写法一：`Counter` + 按频降序（推荐）

```python
from collections import Counter


class Solution:
    def frequencySort(self, s: str) -> str:
        cnt = Counter(s)
        return "".join(ch * c for ch, c in sorted(cnt.items(), key=lambda x: -x[1]))
```

### 写法二：桶排序（频数为桶下标）

```python
from collections import Counter


class Solution:
    def frequencySort(self, s: str) -> str:
        cnt = Counter(s)
        buckets: list[list[str]] = [[] for _ in range(len(s) + 1)]
        for ch, c in cnt.items():
            buckets[c].append(ch)
        res = []
        for f in range(len(s), 0, -1):
            for ch in buckets[f]:
                res.append(ch * f)
        return "".join(res)
```

**要点**：同一频数多个字符时，桶内顺序任意，符合题意。

## ⚠️ 常见陷阱

1. **相同频率时的顺序**  
   - 题目不要求字母序，不要多此一举按字符排序（除非面试官要求）。

2. **只排序字符不展开**  
   - 结果是「每个字符重复 c 次」拼接，不是只输出一个字符。

3. **与 347 对比**  
   - [347. 前 K 个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/) 要**前 k 个**；本题要**全体**按频排列成字符串。
