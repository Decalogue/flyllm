# 438. 找到字符串中所有字母异位词

**标签**：`滑动窗口` `哈希` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
from collections import Counter


class Solution:
    def findAnagrams(self, s: str, p: str) -> list[int]:
        need = Counter(p)
        win = Counter()
        valid = 0
        req = len(need)
        ans = []
        left = 0
        for right, c in enumerate(s):
            win[c] += 1
            if c in need and win[c] == need[c]:
                valid += 1
            while right - left + 1 >= len(p):
                if valid == req:
                    ans.append(left)
                d = s[left]
                win[d] -= 1
                if d in need and win[d] < need[d]:
                    valid -= 1
                left += 1
        return ans
```
