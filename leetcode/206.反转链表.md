# 206. 反转链表

**标签**：`链表` `递归` `简单`  
**分类**：链表、双指针  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表的头节点。

### 示例 1

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

### 示例 2

```
输入：head = [1,2]
输出：[2,1]
```

### 示例 3

```
输入：head = []
输出：[]
```

### 提示

- 链表中节点的数目范围是 `[0, 5000]`
- `-5000 <= Node.val <= 5000`

**进阶**：可以迭代或递归两种方式完成，能否用 O(n) 时间、O(1) 额外空间（迭代）实现？

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**迭代（推荐）**：用 `pre`、`cur` 两指针，每次把 `cur.next` 指向前驱 `pre`，再 `pre, cur = cur, cur.next`（需先用 `tmp` 存 `cur.next` 避免断链）。遍历完后 `pre` 为新头。

**递归**：先递归反转 `head.next` 之后的链表，得到新头；再把 `head.next.next = head`、`head.next = None`，返回新头。

**关键词**：`双指针` `pre/cur` `先存 next 再改指向`

### 复杂度速记

- **迭代**：时间 O(n)，空间 O(1)。
- **递归**：时间 O(n)，空间 O(n)（栈深度）。

### 记忆口诀

「pre 在前 cur 在后，tmp 存 next 防断链；cur.next 指 pre，再一起前移。」

---

## 解题思路

### 核心思想

- **迭代**：从左到右遍历，把「当前节点的 next」从指向后继改为指向前驱。需要记录前驱 `pre`、当前 `cur`，以及在下一次移动前保存 `cur.next`（否则改完 `cur.next` 就找不到原后继）。
- **递归**：假设已能反转 `head.next` 为头的子链表并返回新头 `new_head`；再把 `head` 接到反转后链表的末尾（即让 `head.next.next = head`），并断开 `head.next`，返回 `new_head`。边界：`head` 为空或仅一个节点时直接返回 `head`。

### 算法步骤（迭代）

1. `pre = None`，`cur = head`。
2. 当 `cur` 非空：
   - `tmp = cur.next`（保存后继）；
   - `cur.next = pre`（反转当前边）；
   - `pre, cur = cur, tmp`（前移）。
3. 返回 `pre`（新头）。

---

## 代码实现

### 迭代（推荐，O(1) 空间）

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        pre = None
        cur = head
        while cur:
            tmp = cur.next
            cur.next = pre
            pre = cur
            cur = tmp
        return pre
```

**代码要点**：

- ✅ 必须先 `tmp = cur.next`，再改 `cur.next = pre`，否则会丢后续链表。
- ✅ 循环结束后 `cur` 为 `None`，新头是 `pre`；空链表时 `head` 为 `None`，循环不执行，返回 `pre` 即 `None`，正确。
- ✅ 只用了 `pre`、`cur`、`tmp` 几个指针，空间 O(1)。

### 递归（O(n) 栈空间）

```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        new_head = self.reverseList(head.next)
        head.next.next = head
        head.next = None
        return new_head
```

**要点**：先递归反转 `head.next` 为头的子链表，得到新头 `new_head`；再把 `head` 接到「反转后链表」的末尾（原 `head.next` 变成新链最后一个节点，其 `next` 设为 `head`），最后断开 `head.next`。

---

## 复杂度分析

| 写法 | 时间 | 空间 | 说明 |
|------|------|------|------|
| 迭代 | O(n) | O(1) | 遍历一遍，仅用常数额外指针。 |
| 递归 | O(n) | O(n) | 递归深度为链表长度。 |

---

## 易错点

- **忘记存 next**：改 `cur.next = pre` 前必须先 `tmp = cur.next`，否则无法移动到下一个节点。
- **返回值**：迭代时新头是最后的 `pre`，不是 `cur`（循环结束时 `cur` 为 `None`）。
- **空链表**：`head is None` 时迭代不进入循环，`pre` 仍为 `None`，返回 `None` 即可；递归中 `not head` 直接返回 `head`。
