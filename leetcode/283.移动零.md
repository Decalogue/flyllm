# LeetCode 283. 移动零

**题号**: 283
**难度**: Easy
**标签**: 数组, 双指针, 原地算法

---

## 🎯 题目描述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**注意**: 必须在原数组上操作，不能拷贝额外的数组。

**示例 1**:
```python
输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]
```

**示例 2**:
```python
输入: nums = [0]
输出: [0]
```

**约束条件**:
- `1 <= nums.length <= 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`

---

## 💡 解题思路

### 方法1：双指针交换法（推荐）

**核心思想**：使用快慢指针，在一次遍历中完成移动

**算法步骤**:
1. 初始化慢指针 `slow = 0`，指向可以放置非零元素的位置
2. 快指针 `fast` 遍历整个数组
3. 当 `fast` 指向非零元素时：
   - 交换 `nums[slow]` 和 `nums[fast]`
   - `slow` 指针向前移动（指向下一个可以放置的位置）
4. 遍历完成后，所有非零元素都在 `slow` 左侧，右侧自动填充为0

**关键点**:
- fast 负责"扫描"，slow 负责"放置"
- 当 slow == fast 时，交换无影响（自己和自己交换）
- 只进行一次遍历，时间复杂度 O(n)
- 原地操作，空间复杂度 O(1)

### 方法2：覆盖法（两次遍历）

**算法步骤**:
1. 第一次遍历：将所有非零元素按顺序覆盖到数组前部
2. 第二次遍历：将剩余位置填充为0

优点：逻辑更直观
缺点：需要两次遍历

## 📝 Python 实现

### 实现1：双指针交换（最优解）

```python
from typing import List

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        双指针交换法 - 最优解
        时间复杂度: O(n)
        空间复杂度: O(1)
        """
        slow = 0  # 指向下一个可以放置非零元素的位置

        # fast遍历整个数组
        for fast in range(len(nums)):
            # 当 fast 指向非零元素时进行交换
            if nums[fast] != 0:
                # 交换 fast 和 slow 位置的元素
                nums[slow], nums[fast] = nums[fast], nums[slow]
                # slow前移
                slow += 1

        # 无需显式返回，nums已被修改
```

### 实现2：覆盖法（两次遍历）

```python
from typing import List

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        覆盖法 - 两次遍历
        时间复杂度: O(n)
        空间复杂度: O(1)
        """
        # 第一次遍历：将所有非零元素移动到前面
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow] = nums[fast]
                slow += 1

        # 第二次遍历：将剩余位置填充为0
        for i in range(slow, len(nums)):
            nums[i] = 0
```

|---------|-----------|-----------|----------|---------|
| 双指针交换 | O(n) | O(1) | 1次 | ✅ 强烈推荐 |
| 覆盖法 | O(n) | O(1) | 2次 | ✅ 推荐 |
