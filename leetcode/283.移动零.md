# 283. 移动零

**标签**：`数组` `双指针` `原地` `简单`  
**分类**：数组与字符串（Hot 100）  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给定一个数组 `nums`，编写一个函数将所有 **`0` 移动到数组的末尾**，同时**保持非零元素的相对顺序**。

**注意**：必须在**原数组**上操作，不能使用额外数组拷贝整体数据（常数额外空间内完成）。

### 示例 1

```
输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]
解释: 需注意原地修改，函数无返回值
```

### 示例 2

```
输入: nums = [0]
输出: [0]
```

### 提示

- `1 <= nums.length <= 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：用**慢指针**表示「下一个非零该放的位置」，**快指针**扫描；遇到非零就与慢指针处交换（或先写入再补零），一趟完成。

**关键词**：`快慢指针` `原地` `相对顺序` `分区思想`

### 复杂度速记

- **时间**：O(n) 一次或两次扫描；**空间**：O(1) 额外。

### 记忆口诀

「快扫非零往前挪，慢指针跟着占位；交换版一趟，覆盖版两趟。」

---

## 解题思路

### 核心思想（交换法）

把数组看成两段：**左侧**全是「已就位的非零」，**右侧**待整理。`slow` 始终指向下一个应放非零的下标；`fast` 从左到右走，遇到非零就把该元素换到 `slow`，再 `slow += 1`。这样非零的相对顺序与原来一致（每个非零只被换到更左的空位，顺序未被破坏）。

### 算法步骤（交换）

1. `slow = 0`。  
2. `for fast in range(n)`：若 `nums[fast] != 0`，交换 `nums[slow]` 与 `nums[fast]`，`slow += 1`。  
3. 结束后 `[slow..)` 自然为 0（或被换过去的 0 挤到后面）。

### 覆盖法（两次遍历）

第一次把非零依次写到 `0..slow-1`，第二次把 `slow..n-1` 置 0。思路更直白，常数略大。

---

## 代码实现

### 写法一：双指针交换（推荐，单趟）

```python
class Solution:
    def moveZeroes(self, nums: list[int]) -> None:
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

### 写法二：覆盖 + 填零（双趟）

```python
class Solution:
    def moveZeroes(self, nums: list[int]) -> None:
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow] = nums[fast]
                slow += 1
        for i in range(slow, len(nums)):
            nums[i] = 0
```

## ⚠️ 常见陷阱

1. **新开数组拼 `[非零] + [0]`**  
   - 违反「原地」要求（除非题目明确允许）。

2. **`slow == fast` 时交换**  
   - 合法，等价于不动；不必特判。

3. **与「移除元素」混淆**  
   - [27. 移除元素](https://leetcode.cn/problems/remove-element/) 是删 val、返回新长度；本题要把 0 **留在后面**，语义都是「快指针找保留元素、慢指针写位置」，可对照模板。

| 方法 | 时间 | 额外空间 | 遍数 |
|------|------|----------|------|
| 交换 | O(n) | O(1) | 1 |
| 覆盖+填零 | O(n) | O(1) | 2 |

## 总结

移动零是**原地双指针分区**的基础题：交换法一趟完成，与「慢指针指写入位置、快指针扫描」模板一致，可迁移到移除元素、去重、荷兰国旗等题。
