# 238. 除自身以外数组的乘积

**标签**：`数组` `前缀和` `中等`  
**分类**：数组与字符串（Hot 100）  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给定整数数组 `nums`，返回数组 `answer`，使得 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外**其余各元素的乘积**。

题目数据**保证**数组中任意元素全部乘积不会超出 32 位整数范围。

**进阶**：在 **O(n)** 时间复杂度内完成本题，且**不能使用除法**；**O(1)** 的额外空间复杂度（输出数组**不计入**额外空间）。

### 示例 1

```
输入: nums = [1,2,3,4]
输出: [24,12,8,6]
解释: 除自己外乘积分别为 2*3*4, 1*3*4, 1*2*4, 1*2*3
```

### 示例 2

```
输入: nums = [-1,1,0,-3,3]
输出: [0,0,9,0,0]
```

### 提示

- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`
- 保证任意前缀、后缀的乘积在 32 位整数范围内

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：`answer[i] = 左侧全体乘积 × 右侧全体乘积`；先**从左到右**累乘写入 `answer[i]`，再**从右到左**用变量累乘乘回 `answer[i]`。

**关键词**：`前缀积` `后缀积` `两遍扫描` `O(1) 额外`

### 复杂度速记

- **时间**：O(n)；**额外空间**：O(1)（输出数组 `answer` 可当工作区，不计入）。

### 记忆口诀

「从左扫：answer 先装左积；从右扫：乘上右积一遍成。」

---

## 解题思路

### 核心思想

对下标 `i`，所求为  
\(\prod_{j \neq i} nums[j] = \left(\prod_{j<i} nums[j]\right) \times \left(\prod_{j>i} nums[j]\right)\)。

第一遍：`ans[i]` 先存**前缀积** \(\prod_{j<i} nums[j]\)（从左累乘 `p`）。  
第二遍：用 `p` 从右累乘**后缀**，并 `ans[i] *= p`，即乘上 \(\prod_{j>i} nums[j]\)。

全程除输出外只需常数个变量。

### 为何不能用除法？

数组中可能有 **0**，「总积 / nums[i]」会失效或需大量分支；且进阶明确要求**禁止除法**。

### 算法步骤

1. `ans = [1]*n`，`p = 1`。  
2. `i = 0..n-1`：`ans[i] = p`，`p *= nums[i]`。  
3. `p = 1`，`i = n-1..0`：`ans[i] *= p`，`p *= nums[i]`。  
4. 返回 `ans`。

---

## 代码实现

```python
class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        n = len(nums)
        ans = [1] * n
        p = 1
        for i in range(n):
            ans[i] = p
            p *= nums[i]
        p = 1
        for i in range(n - 1, -1, -1):
            ans[i] *= p
            p *= nums[i]
        return ans
```

---

## ⚠️ 常见陷阱

1. **用总乘积除以 `nums[i]`**  
   - 有 0 时错误；且通常违反「禁止除法」要求。

2. **额外开两个数组存前缀/后缀**  
   - 时间仍 O(n)，但**额外空间 O(n)**，不满足进阶 O(1)。

3. **第一遍就把 `ans[i]` 写成最终答案**  
   - 需分清：第一遍只填**左积**，第二遍再乘**右积**。

---

## 复杂度分析

- **时间复杂度**：O(n)，两遍线性扫描。  
- **空间复杂度**：O(1) 额外（仅若干变量 + 题目要求的输出数组）。

本题是**前缀积 + 后缀积**的模板：两遍扫描、输出数组复用为工作区，即可在 O(n) 时间、O(1) 额外空间下完成，面试常追问「为何不能除法」与「0 的处理」。
