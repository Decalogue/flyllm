# 78. 子集

**标签**：`回溯` `位运算` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 代码实现（迭代）

```python
class Solution:
    def subsets(self, nums: list[int]) -> list[list[int]]:
        res = [[]]
        for x in nums:
            res += [s + [x] for s in res]
        return res
```

---

## 复杂度

- **时间**：O(n·2^n)，**空间**：O(n·2^n)
