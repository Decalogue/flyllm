# 160. 相交链表

**标签**：`链表` `双指针` `简单`  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 代码实现

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def getIntersectionNode(
        self, headA: ListNode, headB: ListNode
    ) -> ListNode | None:
        a, b = headA, headB
        while a is not b:
            a = a.next if a else headB
            b = b.next if b else headA
        return a
```

---

## 复杂度

- **时间**：O(m+n)，**空间**：O(1)
