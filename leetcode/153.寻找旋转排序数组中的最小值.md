# 153. 寻找旋转排序数组中的最小值

**标签**：`二分查找` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
class Solution:
    def findMin(self, nums: list[int]) -> int:
        lo, hi = 0, len(nums) - 1
        while lo < hi:
            mid = (lo + hi) // 2
            if nums[mid] > nums[hi]:
                lo = mid + 1
            else:
                hi = mid
        return nums[lo]
```

---

## 复杂度

- O(log n)
