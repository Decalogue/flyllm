# 134. 加油站

**标签**：`贪心` `数组` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
class Solution:
    def canCompleteCircuit(self, gas: list[int], cost: list[int]) -> int:
        if sum(gas) < sum(cost):
            return -1
        total = start = 0
        for i in range(len(gas)):
            total += gas[i] - cost[i]
            if total < 0:
                total = 0
                start = i + 1
        return start
```

---

## 复杂度

- O(n)
