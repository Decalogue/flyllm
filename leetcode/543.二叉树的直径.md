# 543. 二叉树的直径

**标签**：`树` `简单`  
**难度**：⭐ 简单  

---

## 代码

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def diameterOfBinaryTree(self, root: TreeNode | None) -> int:
        self.ans = 0

        def dep(n: TreeNode | None) -> int:
            if not n:
                return 0
            L, R = dep(n.left), dep(n.right)
            self.ans = max(self.ans, L + R)
            return max(L, R) + 1

        dep(root)
        return self.ans
```
