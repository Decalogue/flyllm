# 122. 买卖股票的最佳时机 II

**标签**：`贪心` `数组` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        return sum(max(0, prices[i] - prices[i - 1]) for i in range(1, len(prices)))
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(1)
