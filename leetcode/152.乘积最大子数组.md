# 152. 乘积最大子数组

**标签**：`动态规划` `数组` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
class Solution:
    def maxProduct(self, nums: list[int]) -> int:
        mx = mn = ans = nums[0]
        for x in nums[1:]:
            t = (mx * x, mn * x, x)
            mx, mn = max(t), min(t)
            ans = max(ans, mx)
        return ans
```

---

## 复杂度

- O(n)
