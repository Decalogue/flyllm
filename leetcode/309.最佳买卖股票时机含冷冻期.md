# 309. 最佳买卖股票时机含冷冻期

**标签**：`动态规划` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        hold, sold, rest = float("-inf"), 0, 0
        for p in prices:
            hold, sold, rest = (
                max(hold, rest - p),
                hold + p,
                max(rest, sold),
            )
        return max(sold, rest)
```
