# 455. 分发饼干

**标签**：`贪心` `简单`  
**难度**：⭐ 简单  

---

## 代码

```python
class Solution:
    def findContentChildren(self, g: list[int], s: list[int]) -> int:
        g.sort()
        s.sort()
        i = 0
        for x in s:
            if i < len(g) and x >= g[i]:
                i += 1
        return i
```
