# 72. 编辑距离

**标签**：`字符串` `动态规划` `困难`  
**分类**：双串前缀 DP、最小编辑步数  
**难度**：⭐⭐⭐ 困难  
**频率**：🔥🔥 高频

---

## 题目描述

给你两个字符串 `word1` 和 `word2`，返回将 `word1` 转换成 `word2` 所使用的**最少操作数**。

你可以进行以下三种操作（每次计 **1** 步）：

1. **插入**一个字符；  
2. **删除**一个字符；  
3. **替换**一个字符。

### 示例 1

```
输入：word1 = "horse", word2 = "ros"
输出：3
解释：
horse -> rorse（将 'h' 替换为 'r'）
rorse -> rose（删除 'r'）
rose -> ros（删除 'e'）
```

### 示例 2

```
输入：word1 = "intention", word2 = "execution"
输出：5
```

### 示例 3

```
输入：word1 = "", word2 = ""
输出：0
```

### 提示

- `0 <= word1.length, word2.length <= 500`
- `word1` 和 `word2` 由小写英文字母组成

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：`dp[i][j]` 表示 `word1` 的前 `i` 个字符变成 `word2` 的前 `j` 个字符的**最少操作数**；末位相同则 `dp[i][j]=dp[i-1][j-1]`；否则在三类操作上取最优：**删 / 插 / 换** 各对应 `dp[i-1][j]`、`dp[i][j-1]`、`dp[i-1][j-1]` 加 1。

**关键词**：`双串前缀 DP` `删插换` `边界一行一列`

### 复杂度速记

- **时间**：O(m·n)。  
- **空间**：O(m·n) 二维表；可滚为 **O(n)** 一行（需保留左上对角 `prev`）。

### 记忆口诀

「相等抄对角；不等三选一，各加一步。」

---

## 解题思路

### 状态定义

- `dp[i][j]`：`word1[0:i]`（长度为 `i` 的前缀）变为 `word2[0:j]` 的最少操作数。  
- 下标约定：`i ∈ [0, m]`，`j ∈ [0, n]`，`m = len(word1)`，`n = len(word2)`。

### 边界（空串）

- `dp[i][0] = i`：把长度为 `i` 的串变成空串，只能 **i 次删除**。  
- `dp[0][j] = j`：空串变成长度 `j`，只能 **j 次插入**。

### 转移

- 若 **`word1[i-1] == word2[j-1]`**：末位已对齐，**无需对末位操作**，`dp[i][j] = dp[i-1][j-1]`。  
- 否则末位至少要动一次，取三种代价的最小再加 1：  
  - **删** `word1` 最后一个：`dp[i-1][j] + 1`；  
  - **插** 入 `word2` 的最后一个（等价于在 `word1` 末尾补）：`dp[i][j-1] + 1`；  
  - **换** 末位使相等：`dp[i-1][j-1] + 1`。

\[
dp[i][j] = 1 + \min\big(dp[i-1][j],\, dp[i][j-1],\, dp[i-1][j-1]\big)
\]

### 与 LCS 的对比（口述）

[1143. 最长公共子序列](1143.最长公共子序列.md) 在不等时是 **`max` 两方向**，因为子序列可跳过字符；编辑距离在不等时要付 **1 步代价** 并在 **删 / 插 / 换** 间取 **min**，模型不同但都是「双串前缀」填表。

## 代码实现

### 写法一：二维 DP（推荐理解与面试默写）

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(m + 1):
            dp[i][0] = i
        for j in range(n + 1):
            dp[0][j] = j
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    dp[i][j] = 1 + min(
                        dp[i - 1][j],      # 删除 word1[i-1]
                        dp[i][j - 1],      # 插入（对齐 word2[j-1]）
                        dp[i - 1][j - 1],  # 替换
                    )
        return dp[m][n]
```

### 写法二：一维滚动（O(n) 空间）

内层 `j` **从小到大**；处理 `dp[j]` 前先保存「左上」`diag = dp[j]`（更新前的 `dp[j]` 即上一行的 `dp[i-1][j-1]`），或使用临时变量轮换。

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = list(range(n + 1))
        for i in range(1, m + 1):
            diag = dp[0]
            dp[0] = i
            for j in range(1, n + 1):
                nd = dp[j]
                if word1[i - 1] == word2[j - 1]:
                    dp[j] = diag
                else:
                    dp[j] = 1 + min(dp[j], dp[j - 1], diag)
                diag = nd
        return dp[n]
```

## ⚠️ 常见陷阱

1. **边界只初始化一行**  
   - 必须同时初始化 **`dp[i][0]`** 与 **`dp[0][j]`**，否则空串情况错。

2. **相等时多加了 1**  
   - 末位相同应 **直接继承** `dp[i-1][j-1]`，不要再 `+1`。

3. **三种操作的含义搞反**  
   - `dp[i-1][j]`：已少处理 `word1` 一个字符 → **删掉** `word1` 末位；`dp[i][j-1]`：**插入**以拉长 `word1` 对齐 `word2` 末位。

4. **与「最长公共子序列」转移混用**  
   - LCS 用 `max`；编辑距离在不等时用 **`min` + 1** 与三个前驱。

## 复杂度分析

- **时间复杂度**：O(m·n)。  
- **空间复杂度**：O(m·n)（二维）或 O(min(m, n))（可对较短串长度滚动，减少一行长度）。

## 总结

72 是**双串前缀**经典题：行列表征 `word1` 前缀、列表征 `word2` 前缀，**相等抄对角**，**不等**在**删、插、换**三种前驱上取最小再加一。面试先写二维 + 完整边界，追问再压缩为一行并说明 **`diag` 保存左上**。
