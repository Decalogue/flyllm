# 55. 跳跃游戏

**标签**：`贪心` `数组` `中等`  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 解题思路

维护**最远可达下标** `reach`；遍历 `i`，若 `i > reach` 失败；否则 `reach = max(reach, i+nums[i])`。

---

## 代码实现

```python
class Solution:
    def canJump(self, nums: list[int]) -> bool:
        reach = 0
        for i, x in enumerate(nums):
            if i > reach:
                return False
            reach = max(reach, i + x)
            if reach >= len(nums) - 1:
                return True
        return True
```

---

## 复杂度

- **时间**：O(n)，**空间**：O(1)
