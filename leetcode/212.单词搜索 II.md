# 212. 单词搜索 II

**标签**：`Trie` `回溯` `困难`  
**难度**：⭐⭐⭐ 困难  

---

## 说明

将 `words` 建 **Trie**，网格 DFS，走到结点带 `#` 的单词则加入答案并剪枝。

---

## 代码（框架）

```python
class Solution:
    def findWords(self, board: list[list[str]], words: list[str]) -> list[str]:
        trie = {}
        for w in words:
            n = trie
            for c in w:
                n = n.setdefault(c, {})
            n["#"] = w
        m, n = len(board), len(board[0])
        ans = []

        def dfs(i: int, j: int, node: dict) -> None:
            if "#" in node:
                ans.append(node.pop("#"))
            if i < 0 or i >= m or j < 0 or j >= n:
                return
            c = board[i][j]
            if c not in node:
                return
            board[i][j] = "#"
            nxt = node[c]
            for di, dj in ((1, 0), (-1, 0), (0, 1), (0, -1)):
                dfs(i + di, j + dj, nxt)
            board[i][j] = c
            if not nxt:
                node.pop(c)

        for i in range(m):
            for j in range(n):
                dfs(i, j, trie)
        return ans
```

---

## 注意

边界与 Trie 删除需按题解细调；以上为常见写法思路。
