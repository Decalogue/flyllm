# 1143. 最长公共子序列

**标签**：`字符串` `动态规划` `中等`  
**分类**：DP、字符串  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给定两个字符串 `text1` 和 `text2`，返回这两个字符串的**最长公共子序列（LCS）**的长度。若不存在公共子序列，返回 `0`。

- **子序列**：由原字符串在不改变字符相对顺序的前提下，删除若干字符（可不删）得到的新字符串。例如 `"ace"` 是 `"abcde"` 的子序列，`"aec"` 不是。
- **公共子序列**：同时是两个字符串的子序列的字符串。

### 示例 1

```
输入：text1 = "abcde", text2 = "ace"
输出：3
解释：最长公共子序列是 "ace"，长度为 3。
```

### 示例 2

```
输入：text1 = "abc", text2 = "abc"
输出：3
解释：最长公共子序列是 "abc"，长度为 3。
```

### 示例 3

```
输入：text1 = "abc", text2 = "def"
输出：0
解释：两个字符串没有公共子序列，返回 0。
```

### 提示

- `1 <= text1.length, text2.length <= 1000`
- `text1` 和 `text2` 仅由小写英文字符组成

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**二维 DP**：`dp[i][j]` 表示 `text1[0:i]` 与 `text2[0:j]` 的 LCS 长度。若 `text1[i-1] == text2[j-1]`，则 `dp[i][j] = dp[i-1][j-1] + 1`；否则 `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`。答案为 `dp[m][n]`。

**关键词**：`二维DP` `双串前缀` `相等则+1否则取max`

### 复杂度速记

- **时间**：O(m·n)——两重循环填表。
- **空间**：O(m·n)；可滚动数组优化为 O(min(m,n))。

### 记忆口诀

「dp[i][j] 表两串前缀 LCS；末位同则左上+1，不同则左、上取大。」

---

## 解题思路

### 核心思想

- **状态**：`dp[i][j]` = `text1[0:i]` 与 `text2[0:j]` 的最长公共子序列长度（下标 i、j 表示长度，即 `text1[i-1]`、`text2[j-1]` 为当前末位）。
- **转移**：
  - 若 `text1[i-1] == text2[j-1]`：当前位可同时选入 LCS，`dp[i][j] = dp[i-1][j-1] + 1`。
  - 否则：至少有一个不在当前 LCS 里，`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`（舍 text1 末位 或 舍 text2 末位）。
- **边界**：`dp[0][*]` 与 `dp[*][0]` 为 0（空串与任意串的 LCS 长度为 0）。

### 算法步骤

1. 令 `m = len(text1)`, `n = len(text2)`，建立 `dp[(m+1)][(n+1)]`，初始为 0。
2. `i` 从 1 到 m，`j` 从 1 到 n：
   - 若 `text1[i-1] == text2[j-1]`：`dp[i][j] = dp[i-1][j-1] + 1`。
   - 否则：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`。
3. 返回 `dp[m][n]`。

---

## 代码实现

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        # dp[i][j] 表示 text1[0:i] 和 text2[0:j] 的最长公共子序列的长度
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

        return dp[m][n]
```

**代码要点**：

- ✅ `dp` 开 `(m+1) x (n+1)`，下标 0 表示空串，避免边界特判。
- ✅ 比较的是 `text1[i-1]` 与 `text2[j-1]`，对应长度为 i、j 的前缀的最后一个字符。
- ✅ 相等时用左上 `dp[i-1][j-1]`；不等时取 `dp[i-1][j]` 与 `dp[i][j-1]` 的较大值。

## 复杂度分析

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m·n) | 双重循环填满 dp 表。 |
| 空间 | O(m·n) | 二维 dp 数组。可改为两行或一维滚动，空间 O(min(m,n))。 |

---

## 易错点

- **下标**：`dp[i][j]` 对应的是 `text1[0:i]`（长度为 i），所以当前字符是 `text1[i-1]`、`text2[j-1]`，不要写成 `text1[i]`、`text2[j]`。
- **初始化**：`dp[0][j]` 和 `dp[i][0]` 已为 0，无需再写；若从 0 开始循环要小心越界。
- **与最长公共子串区别**：子串必须连续，子序列可不连续；子串的 DP 是「相等才加 1，不等置 0」，LCS 是「不等取 max」。
