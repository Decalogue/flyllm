# 714. 买卖股票的最佳时机含手续费

**标签**：`贪心` `数组` `动态规划` `中等`  
**分类**：股票系列、状态机  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 高频

---

## 题目描述

给定整数数组 `prices`，其中 `prices[i]` 表示某支股票第 `i` 天的价格；非负整数 `fee` 表示**每笔交易**需要支付的手续费（卖出时扣费，具体扣费时机以状态定义为准，下面解法为**卖出时**付 `fee`）。

你可以**多次**买入卖出同一支股票，但不能同时参与多笔交易（必须先卖掉再买入）。求能获得的最大利润。

### 示例 1

```
输入：prices = [1,3,2,8,4,9], fee = 2
输出：8
解释：买入价为 1，卖出价为 8，利润 8-1-2=5；再买入 4，卖出 9，利润 9-4-2=3；共 8。
```

### 示例 2

```
输入：prices = [1,3,7,5,10,3], fee = 3
输出：6
```

### 提示

- `1 <= prices.length <= 5 * 10^4`
- `1 <= prices[i] < 5 * 10^4`
- `0 <= fee < 5 * 10^4`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：每天维护两个状态——**手里持股**的最大收益 `hold`、**手里只有现金**的最大收益 `cash`；卖出时在变现里扣掉 `fee`。

**关键词**：`状态机` `hold/cash` `卖出扣手续费`

### 复杂度速记

- **时间**：O(n) 扫一遍。  
- **空间**：O(1) 两个变量。

### 记忆口诀

「持股看买价，空仓看卖价减 fee；每天取 max 更新。」

---

## 解题思路

### 核心思想

任意一天结束时，只有两种可能：**持有股票**或**不持有**。用 `hold`、`cash` 表示在这两种情况下，到当天为止能达到的最大利润（相对初始现金 0）。

- **买入**：现金变股票，`hold = max(hold, cash - prices[i])`（保持继续持有或今天买入）。  
- **卖出**：股票变现金并付手续费，`cash = max(cash, hold + prices[i] - fee)`（继续空仓或今天卖出）。

初始：`hold = -∞`（或理解为不可能的状态）、`cash = 0`。答案为最后的 `cash`。

### 为什么卖出扣 fee？

本题常见约定：**完成一笔交易（卖）时**支付 `fee`。在转移方程里体现在 `hold + p - fee`：把股票按 `p` 卖掉，到手现金减 `fee`。

### 算法步骤

1. `hold = -inf`，`cash = 0`。  
2. 对每个价格 `p`：  
   - `hold = max(hold, cash - p)`  
   - `cash = max(cash, hold + p - fee)`  
   - 注意：同一天更新时应用**上一轮**的 `hold`/`cash`，实现上应用**同时更新**的临时变量或按顺序写（见下方代码顺序）。  
3. 返回 `cash`。

**实现细节**：若先算 `hold` 再算 `cash`，当天用的 `hold` 已是更新后的，会允许「同一天又买又卖」。应使用**上一日**的两个状态，典型写法：

```text
new_hold = max(hold, cash - p)
new_cash = max(cash, hold + p - fee)
hold, cash = new_hold, new_cash
```

或在一行元组赋值中，右侧全部用旧值（Python 中先算右边再赋左边，下面「写法一」的两行顺序在数学上等价于用旧 `hold` 更新 `cash`，需验证）。

标准正确写法：

```python
for p in prices:
    hold, cash = max(hold, cash - p), max(cash, hold + p - fee)
```

在 Python 中，`hold, cash = A, B` 的 `B` 里的 `hold` 是**赋值前**的旧 `hold`，`A` 里的 `cash` 是旧 `cash`。因此：  
- 第一个 `max(hold, cash - p)` 用的是旧 hold、旧 cash ✓  
- 第二个 `max(cash, hold + p - fee)` 里 `cash` 是旧 cash，`hold` 是旧 hold ✓  

故**一行双赋值**在 Python 中是正确的。

## 代码实现

### 写法一：双状态（推荐）

```python
class Solution:
    def maxProfit(self, prices: list[int], fee: int) -> int:
        hold, cash = float("-inf"), 0
        for p in prices:
            hold, cash = max(hold, cash - p), max(cash, hold + p - fee)
        return cash
```

### 写法二：显式临时变量（便于口述）

```python
class Solution:
    def maxProfit(self, prices: list[int], fee: int) -> int:
        hold, cash = float("-inf"), 0
        for p in prices:
            nh = max(hold, cash - p)
            nc = max(cash, hold + p - fee)
            hold, cash = nh, nc
        return cash
```

## ⚠️ 常见陷阱

1. **手续费加在买入还是卖出**  
   - 本题状态转移与「卖出付 fee」一致；若题面改为买入付 fee，应对应改转移（等价于调整价格，见思考题）。

2. **`hold` 初值**  
   - 用 `-inf` 表示尚未买过；若用 `0` 会错误允许「凭空持股」。

3. **与 122（无手续费）对比**  
   - `fee = 0` 时，`cash = max(cash, hold + p)` 即经典**无限次交易**贪心/状态机。
