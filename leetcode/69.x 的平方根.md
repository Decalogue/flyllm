# 69. x 的平方根

**标签**：`二分查找` `数学` `简单`  
**难度**：⭐ 简单  
**频率**：🔥🔥 高频

---

## 代码实现

```python
class Solution:
    def mySqrt(self, x: int) -> int:
        lo, hi = 0, x
        while lo <= hi:
            mid = (lo + hi) // 2
            if mid * mid <= x < (mid + 1) * (mid + 1):
                return mid
            if mid * mid < x:
                lo = mid + 1
            else:
                hi = mid - 1
        return hi
```

---

## 复杂度

- **时间**：O(log x)
