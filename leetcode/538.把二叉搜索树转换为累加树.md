# 538. 把二叉搜索树转换为累加树

**标签**：`BST` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def convertBST(self, root: TreeNode | None) -> TreeNode | None:
        s = 0

        def dfs(n: TreeNode | None) -> None:
            nonlocal s
            if not n:
                return
            dfs(n.right)
            s += n.val
            n.val = s
            dfs(n.left)

        dfs(root)
        return root
```
