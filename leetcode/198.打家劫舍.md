# 198. 打家劫舍

**标签**：`数组` `动态规划` `中等`  
**分类**：线性 DP、状态压缩  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

你是一个专业小偷，计划沿街偷窃。每间房都存放了一定现金；**相邻两间房不能同时被偷**（会触发报警）。

给定一个**非负整数数组** `nums`，`nums[i]` 表示第 `i` 间房的金额。在**不触动报警**的前提下，求一夜能偷到的**最大金额**。

### 示例 1

```
输入：nums = [1,2,3,1]
输出：4
解释：偷第 1 间（金额 1）和第 3 间（金额 3），总额 1 + 3 = 4。
```

### 示例 2

```
输入：nums = [2,7,9,3,1]
输出：12
解释：偷第 1、3、5 间，2 + 9 + 1 = 12。
```

### 提示

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：考虑第 `i` 间房，要么**不偷**（继承前一间最优），要么**偷**（加上 `i-2` 及之前的最优）；`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`。

**关键词**：`线性 DP` `相邻约束` `滚动变量 O(1) 空间`

### 复杂度速记

- **时间**：O(n) 扫一遍。  
- **空间**：O(1) 只需两个变量（或 O(n) 显式 `dp` 数组便于理解）。

### 记忆口诀

「偷这家要搭上家前前最优；不偷就沿用上一家的最优。」

---

## 解题思路

### 状态定义

- `dp[i]`：只考虑前 `i+1` 间房（下标 `0..i`）时，能偷到的**最大金额**。

### 转移方程

对第 `i` 间房（`i >= 1`）：

- **不偷 `i`**：金额为 `dp[i-1]`（前 `i-1` 间已最优）。  
- **偷 `i`**：不能偷 `i-1`，故金额为 `dp[i-2] + nums[i]`。

取较大者：

\[
dp[i] = \max(dp[i-1],\, dp[i-2] + nums[i])
\]

### 边界

- `dp[0] = nums[0]`（只有一间只能偷它）。  
- `i=1` 时：`dp[1] = max(nums[0], nums[1])`，与上式一致（可令 `dp[-1]=0` 统一写）。

### 空间优化

`dp[i]` 只依赖 `dp[i-1]` 与 `dp[i-2]`，用两个变量滚动即可，**O(1) 额外空间**。

---

## 代码实现

### 写法一：滚动变量（推荐，与常见题解一致）

用 `a` 表示「上一间为止的最优」（相当于 `dp[i-2]` 的角色在迭代中会左移），`b` 表示「上上一段结束时的最优」——下面实现里 **`a` 存前两间最优、`b` 存前一间最优** 的滚动更直观：

等价于维护 `prev2 = dp[i-2]`，`prev1 = dp[i-1]`，每来一间 `cur = max(prev1, prev2 + x)`，再滚动。

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        prev2, prev1 = 0, 0
        for x in nums:
            prev2, prev1 = prev1, max(prev1, prev2 + x)
        return prev1
```

### 写法二：`a, b` 双变量（原笔记写法）

初始 `a = b = 0` 表示「尚未考虑任何房间」时，前一段最优为 0。每读一个 `x`：`a, b = b, max(b, a + x)` —— 此时 **`a` 是上一轮的「较旧」最优（相当于到 `i-2`），`b` 是到 `i-1` 的最优**。

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        a = b = 0
        for x in nums:
            a, b = b, max(b, a + x)
        return b
```

### 写法三：显式 `dp` 数组（便于讲清下标）

```python
class Solution:
    def rob(self, nums: list[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0]
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, n):
            dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
        return dp[n - 1]
```

## ⚠️ 常见陷阱

1. **误以为要隔一间必偷一间**  
   - 最优策略可以连续多间不偷，只比较「偷 / 不偷当前」两种继承关系即可。

2. **边界 `n == 1`**  
   - 显式 `dp` 写法需注意只有一间时不要访问 `dp[1]`。

3. **负数金额**  
   - 本题非负；若变种出现负数，仍可用同一转移（或需重新审题是否必须偷满等）。

4. **与环形、树形混淆**  
   - **198** 是**一条线**；环形见 [213. 打家劫舍 II](213.打家劫舍%20II.md)；树形见 [337. 打家劫舍 III](https://leetcode.cn/problems/house-robber-iii/)。

## 复杂度分析

- **时间复杂度**：O(n)，`n = len(nums)`。  
- **空间复杂度**：O(1)（写法一、二）或 O(n)（写法三）。

## 总结

198 是最经典的 **「相邻不能选」线性 DP**：`max(不选当前, 选当前 + 前前最优)`。面试先口述转移再写 **双变量滚动**，时间 O(n)、空间 O(1)；需要画图讲区间时再写 `dp[i]` 数组版。
