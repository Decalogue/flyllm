# 102. 二叉树的层序遍历

**标签**：`BFS` `树` `广度优先搜索` `中等`  
**分类**：二叉树、层序遍历  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给你二叉树的根结点 `root`，返回其结点值的**层序遍历**结果（即**逐层地、从左到右**访问所有结点）。

**层序遍历**：按「深度」分层，同一层从左到右依次访问；通常用**队列**实现，与 DFS 的「一路走到底」不同，BFS 是**一圈一圈向外扩展**。

### 示例 1

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

### 示例 2

```
输入：root = [1]
输出：[[1]]
```

### 示例 3

```
输入：root = []
输出：[]
```

### 提示

- 树中结点数目在范围 `[0, 2000]` 内  
- `-1000 <= Node.val <= 1000`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：队列里只放「当前层及下一层待处理结点」；每轮处理前用 `len(q)` 固定本层个数，依次弹出并把孩子入队，保证**一层一层**输出。

**关键词**：`BFS` `deque` `每层快照 len(q)` `左先于右入队`

### 复杂度速记

- **时间**：O(n)，每个结点入队、出队各常数次。  
- **空间**：O(w)，w 为队列中结点数的最大值，最坏约 **O(n)**（最后一层或近似满层）。

### 记忆口诀

「层序靠队列：本层有几个先数清（`len`），弹完再收下一层。」

---

## 解题思路

### 核心思想

层序遍历本质是**无权图上的 BFS**：从根出发，先访问距离为 1 的结点，再距离为 2，依此类推。二叉树里「距离」即**深度**，同一深度的结点构成一层。

**关键点：**

- 用 **FIFO 队列** 维护「待访问」结点；根先入队。  
- **每轮循环 = 一层**：循环开始前记录 `size = len(q)`，只处理这 `size` 个结点；它们的孩子在本轮末尾入队，自然成为下一层。  
- 若不用 `size` 而是一次只弹一个，会把相邻两层混在一起，无法分组。

### 算法步骤

1. 若 `root` 为空，返回 `[]`。  
2. `q = deque([root])`，`ans = []`。  
3. 当 `q` 非空：  
   - `level = []`，`n = len(q)`。  
   - 重复 `n` 次：弹出队首 `node`，把 `node.val` 加入 `level`；若左右孩子非空则依次入队。  
   - 将 `level` 追加到 `ans`。  
4. 返回 `ans`。

### 为什么选择 BFS？

- **优势**：与「按层输出」的定义一致，代码短、边界清晰。  
- **适用场景**：层序、最短路径（边权为 1）、逐层统计等。  
- **替代方案**：**DFS + 深度**：递归时带层号 `d`，把值 append 到 `ans[d]`；思路不同，同样 O(n)，但需预分配或按深度扩列表，面试两种都应能写。

## 代码实现

### 写法一：BFS + 队列（推荐 ⭐）

```python
from collections import deque


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def levelOrder(self, root: TreeNode | None) -> list[list[int]]:
        if not root:
            return []
        q = deque([root])
        ans: list[list[int]] = []
        while q:
            level: list[int] = []
            for _ in range(len(q)):
                n = q.popleft()
                level.append(n.val)
                if n.left:
                    q.append(n.left)
                if n.right:
                    q.append(n.right)
            ans.append(level)
        return ans
```

**代码要点**：

- ✅ `for _ in range(len(q))` 必须在**进入本层循环时**求值，表示「当前层结点个数」，不能边弹边用变化的 `len(q)` 当层界（除非用哨兵等别的技巧）。  
- ✅ 孩子**先左后右**与「从左到右」一致。  
- ✅ 空树提前返回，避免对空队列误操作。

### 写法二：DFS + 按深度分组（备选）

先序遍历顺序访问，用当前深度 `d` 把值放进 `ans[d]`。

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def levelOrder(self, root: TreeNode | None) -> list[list[int]]:
        ans: list[list[int]] = []

        def dfs(n: TreeNode | None, d: int) -> None:
            if not n:
                return
            if d == len(ans):
                ans.append([])
            ans[d].append(n.val)
            dfs(n.left, d + 1)
            dfs(n.right, d + 1)

        dfs(root, 0)
        return ans
```

**要点**：访问顺序仍是「深度优先」，但按 `d` 分桶后，每一桶内顺序与层序一致（先左子树再右子树在同一层里对应「从左到右」）。

### 执行过程示例

以 `root = [3,9,20,null,null,15,7]` 为例（队列从左到右）：

| 步骤 | 队列（简写） | 本层输出 |
|------|----------------|----------|
| 初值 | `[3]` | — |
| 第 1 层 | 弹出 3，入 9、20 | `[3]` |
| 第 2 层 | 弹出 9、20，入 15、7 | `[9,20]` |
| 第 3 层 | 弹出 15、7 | `[15,7]` |

## ⚠️ 常见陷阱

1. **没有用本层长度固定循环次数**  
   - ❌ `while q` 里只 `popleft` 一次，或用一个全局计数混层。  
   - ✅ 每层开始前 `n = len(q)`，内层循环恰好 `n` 次。

2. **与「单结点扩展」模板混淆**  
   - 若每次只弹出一个结点并扩展孩子，而不按层批量处理，得到的是**扁平 BFS 序列**，不是 `[[层1], [层2], ...]`。分层时必须**先记本层个数再循环**，或使用 DFS 按深度分桶。

3. **空树**  
   - `root is None` 时直接返回 `[]`，不要入队 `None` 再在循环里判空（徒增分支，易写错）。

4. **与「每层一行」类题目的迁移**  
   - [103](https://leetcode.cn/problems/binary-tree-zigzag-level-order-traversal/) 之字形只需在奇数层反转 `level` 或双端插入；[199](https://leetcode.cn/problems/binary-tree-right-side-view/) 每层只保留最后一个值。模板仍是「按层处理」。

---

## 复杂度分析

| 方法 | 时间 | 额外空间 | 说明 |
|------|------|----------|------|
| BFS | O(n) | O(w) ≤ O(n) | w 为队列最大宽度。 |
| DFS 分桶 | O(n) | O(h) | 递归栈 O(h)；`ans` 存全部结点值 O(n)。 |

---

## 关键点

1. **层序 = BFS**：队列 + 每层 `len(q)` 是面试最常考的写法。  
2. **宽度与空间**：满二叉树最后一层约 n/2 个结点，故最坏空间 O(n)。  
3. **变种**：锯齿形层序、每层最大值、最底层最左值等，都在此模板上改「每层收集方式」即可。

## 总结

本题是**二叉树 BFS** 的标准题：**`deque` + 每层用 `len(q)` 控制弹出次数**，即可稳定得到 `List[List[int]]`。进阶需能口述 **DFS 按深度分桶** 的等价思路，并能迁移到锯齿序、右视图等变体。
