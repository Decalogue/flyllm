# 647. 回文子串

**标签**：`字符串` `动态规划` `中等`  
**分类**：中心扩展、区间 DP、回文计数  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给定字符串 `s`，返回 `s` 中**回文子串**的**个数**。

- **子串**：字符串中**连续**的一段；不同位置上的相同内容算作**不同**子串（例如 `"aaa"` 中有多个 `"a"`、`"aa"`）。  
- **回文串**：正读与反读相同。

### 示例 1

```
输入：s = "abc"
输出：3
解释：回文子串为 "a"、"b"、"c"。
```

### 示例 2

```
输入：s = "aaa"
输出：6
解释：六个回文子串分别为 "a"（三处）、"aa"（两处）、"aaa"。
```

### 提示

- `1 <= s.length <= 1000`
- `s` 由小写英文字母组成

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：枚举每个**回文中心**（奇长度用 `expand(i,i)`，偶长度用 `expand(i,i+1)`），向两侧扩展；**每成功扩展一层**就对应**一个新的回文子串**，计数加一。

**关键词**：`中心扩展` `奇偶中心` `计数而非求最长`

### 复杂度速记

- **中心扩展**：时间 **O(n²)**，空间 **O(1)**。  
- **区间 DP**：时间 **O(n²)**，空间 **O(n²)**（可优化为 O(n)）。  
- **Manacher**：时间 **O(n)**，实现较长，可作进阶。

### 记忆口诀

「与第 5 题同款扩展，每扩成一层就 +1。」

---

## 解题思路

### 与 [5. 最长回文子串](5.最长回文子串.md) 的关系

- **5**：只关心**最长**那一段，扩展时更新 `max_len`。  
- **647**：要**所有**回文子串个数；同一套「奇中心 / 偶中心 + 双指针扩展」，在 `while` 里**每轮** `s[l]==s[r]` 就说明 `[l,r]` 是回文，**`ans += 1`**。

每个回文子串有**唯一中心**（奇中心一个下标，偶中心两个下标之间），因此不会被重复计数。

### 方法二：区间布尔 DP（可选）

`dp[i][j]` 表示 `s[i:j+1]` 是否为回文：

- `s[i]==s[j]` 且（长度 ≤ 2 或 `dp[i+1][j-1]`）时为真；  
- 统计所有 `dp[i][j]` 为真的对数。

与 5 题文档中的「最长回文子串」DP 同表结构，本题改为**计数**。

### 进阶

可用 **Manacher** 在 O(n) 内求出每个中心的回文半径，再换算子串个数；面试一般 **中心扩展** 即可。

## 代码实现

### 写法一：中心扩展（推荐）

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        ans = 0

        def expand(l: int, r: int) -> None:
            nonlocal ans
            while l >= 0 and r < n and s[l] == s[r]:
                ans += 1
                l -= 1
                r += 1

        for i in range(n):
            expand(i, i)       # 奇数长度
            expand(i, i + 1)   # 偶数长度
        return ans
```

### 写法二：区间 DP

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        dp = [[False] * n for _ in range(n)]
        cnt = 0
        for j in range(n):
            for i in range(j + 1):
                if s[i] == s[j]:
                    if j - i < 3:
                        dp[i][j] = True
                    else:
                        dp[i][j] = dp[i + 1][j - 1]
                if dp[i][j]:
                    cnt += 1
        return cnt
```

## ⚠️ 常见陷阱

1. **与「最长」混淆**  
   - 本题要**个数**，扩展时每次合法区间都要计数，不能只记最大值。

2. **与「子序列」混淆**  
   - 必须是**连续子串**；子序列不连续，模型不同。

3. **偶回文漏计**  
   - 必须调用 **`expand(i, i + 1)`**，否则会漏掉 `"aa"` 这类偶长度回文。

4. **重复计数**  
   - 每个回文子串对应唯一中心扩展路径，按上述写法**不会**重复计数。

---

## 复杂度分析

- **中心扩展**：共 O(n) 个中心，每个最多扩展 O(n) 步 → **时间 O(n²)**，**空间 O(1)**。  
- **DP**：**时间 O(n²)**，**空间 O(n²)**（二维表）。

## 总结

647 是 **5 题的计数版**：奇 / 偶中心向两侧扩，**每扩一层 `ans += 1`** 即得所有回文子串个数；实现短、常数好。若需与区间 DP 体系统一，可用与 5 相同的布尔 `dp[i][j]` 再统计 `True` 个数。
