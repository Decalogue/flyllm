# 347. 前 K 个高频元素

**标签**：`堆` `哈希` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
import heapq
from collections import Counter


class Solution:
    def topKFrequent(self, nums: list[int], k: int) -> list[int]:
        c = Counter(nums)
        return heapq.nlargest(k, c.keys(), key=c.__getitem__)
```
