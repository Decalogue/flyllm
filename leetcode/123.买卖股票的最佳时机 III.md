# 123. 买卖股票的最佳时机 III

**标签**：`动态规划` `困难`  
**难度**：⭐⭐⭐ 困难  

---

## 思路

最多两笔交易：状态 `buy1/sell1/buy2/sell2` 或 `dp[k][i]`。

---

## 代码

```python
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        b1 = b2 = float("inf")
        s1 = s2 = 0
        for p in prices:
            b1 = min(b1, p)
            s1 = max(s1, p - b1)
            b2 = min(b2, p - s1)
            s2 = max(s2, p - b2)
        return s2
```

---

## 复杂度

- O(n) 时间，O(1) 空间
