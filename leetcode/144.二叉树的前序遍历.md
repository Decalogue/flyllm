# 144. 二叉树的前序遍历

**标签**：`栈` `树` `深度优先搜索` `简单`  
**分类**：二叉树、遍历  
**难度**：⭐ 简单  
**频率**：🔥🔥🔥 高频

---

## 题目描述

给你二叉树的根节点 `root`，返回其节点值的**前序遍历**。

**前序遍历**：先访问**根**结点，再**前序**遍历左子树，再**前序**遍历右子树。顺序记作：**根 → 左 → 右**。

### 示例 1

```
输入：root = [1,null,2,3]
输出：[1,2,3]
```

### 示例 2

```
输入：root = []
输出：[]
```

### 示例 3

```
输入：root = [1]
输出：[1]
```

### 提示

- 树中结点数目范围：`[0, 100]`
- `-100 <= Node.val <= 100`

**进阶**：递归解法很简单，能否用**迭代**完成？

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：递归直接按「根左右」；迭代用栈模拟，因先处理根再左再右，入栈时**先压右再压左**，保证左先出。

**关键词**：`前序` `根左右` `栈先右后左`

### 复杂度速记

- **时间**：O(n)，每个结点访问一次。  
- **空间**：递归 O(h)；迭代栈最坏 O(h)，h 为树高。

### 记忆口诀

「前序根先行，迭代栈里右先进。」

---

## 解题思路

### 核心思想

前序是 **DFS** 的一种访问顺序。递归写法与定义一致。迭代时，栈保存「待访问的子树」：弹出结点访问后，若存在右孩子、左孩子，依次压栈，则下次先弹左孩子，符合前序。

### 算法步骤（迭代）

1. 若 `root` 为空，返回空列表。  
2. 初始化 `res = []`，栈 `st = [root]`。  
3. 当栈非空：弹出栈顶 `n`，将 `n.val` 加入 `res`。  
4. 若 `n.right` 存在，入栈；若 `n.left` 存在，入栈（**先右后左**）。  
5. 重复 3–4 直至栈空，返回 `res`。

### 为什么选择迭代？

- **面试常问**：在禁止递归或栈深度受限场景下需写显式栈。  
- **递归**：代码最短，空间为函数调用栈 O(h)。

## 代码实现

### 写法一：递归（最直观）

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def preorderTraversal(self, root: TreeNode | None) -> list[int]:
        res: list[int] = []

        def dfs(n: TreeNode | None) -> None:
            if not n:
                return
            res.append(n.val)
            dfs(n.left)
            dfs(n.right)

        dfs(root)
        return res
```

**要点**：访问顺序严格为根 → 左 → 右。

### 写法二：迭代 + 栈（推荐应对进阶）

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def preorderTraversal(self, root: TreeNode | None) -> list[int]:
        if not root:
            return []
        res: list[int] = []
        st: list[TreeNode] = [root]
        while st:
            n = st.pop()
            res.append(n.val)
            if n.right:
                st.append(n.right)
            if n.left:
                st.append(n.left)
        return res
```

**要点**：**先压右再压左**，出栈顺序才是「左先于右」。

### 写法三：统一迭代风格（标记法，可选）

部分题解用 `(node, visited)` 或 `null` 标记区分「第一次遇到」与「子树处理完」，写法与中后序统一；前序最简单时一般用写法二即可。

---

## ⚠️ 常见陷阱

1. **入栈顺序反了**  
   - ❌ 先压左再压右 → 会先访问右子树。  
   - ✅ 先压右再压左，保证左子树先扩展。

2. **空树**  
   - 直接返回 `[]`，勿对 `None` 入栈后未判空就访问 `.val`。

3. **与中序、后序混淆**  
   - 中序迭代需一路向左压栈；后序常需「根最后输出」或双栈技巧，**不要**把中序模板套到前序。

---

## 复杂度分析

| 写法 | 时间 | 额外空间 |
|------|------|----------|
| 递归 | O(n) | O(h) 调用栈 |
| 迭代栈 | O(n) | O(h) 栈 |

最坏（链状树）h = n；完全二叉树 h = O(log n)。
