# 32. 最长有效括号

**标签**：`字符串` `栈` `动态规划` `困难`  
**分类**：栈、DP、字符串  
**难度**：⭐⭐⭐ 困难  
**频率**：🔥🔥 中高频

---

## 题目描述

给你一个只包含 `'('` 和 `')'` 的字符串，找出**最长有效**（格式正确且连续）括号**子串**的长度。

- **有效**：左右括号一一匹配，每个左括号都有对应的右括号闭合，例如 `"(()())"`。

### 示例 1

```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

### 示例 2

```
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
```

### 示例 3

```
输入：s = ""
输出：0
```

### 提示

- `0 <= s.length <= 3 * 10^4`
- `s[i]` 为 `'('` 或 `')'`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**栈做法**：栈里存**未匹配左括号的下标**，初始放入 `-1` 表示「有效段起点前一位」。遇 `'('` 压入当前下标；遇 `')'` 先弹栈，若栈非空则当前有效段长度为 `i - stack[-1]`，用其更新答案；若栈空说明多了一个 `')'`，把当前下标压栈作为新的「无效分界」。

**关键词**：`栈存下标` `初始 -1` `弹栈后栈顶为上一无效位置`

### 复杂度速记

- **时间**：O(n)——单遍遍历。
- **空间**：O(n)——栈最坏存 n 个下标。

### 记忆口诀

「左括号压下标，右括号弹栈；栈非空用 i 减栈顶更新长度，栈空则压当前 i。」

---

## 解题思路

### 核心思想

任意一段连续有效括号，等价于：在这段之前有一个「最后一个未匹配的右括号或起始位置」，这段内左右完全匹配。用栈维护「最后一个使当前无法继续向左延伸的位置」的下标（无效分界），则 **当前下标 − 该分界** 就是以当前右端点结尾的有效长度。

### 算法步骤（栈）

1. 若 `s` 为空，返回 0。
2. `stack = [-1]`，`res = 0`。
3. 对 `i, c` 遍历 `s`：
   - `c == '('`：`stack.append(i)`。
   - `c == ')'`：`stack.pop()`；若 `stack` 非空则 `res = max(res, i - stack[-1])`；否则 `stack.append(i)`。
4. 返回 `res`。

---

## 代码实现

### 写法一：栈（推荐）

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        if not s:
            return 0
        res = 0
        stack = [-1]
        for i, c in enumerate(s):
            if c == "(":
                stack.append(i)
            else:
                stack.pop()
                if stack:
                    res = max(res, i - stack[-1])
                else:
                    stack.append(i)
        return res
```

**代码要点**：

- ✅ 栈初始为 `[-1]`：这样第一个有效段 `"()"` 的长度为 `i - (-1) = 2`。
- ✅ 弹栈后栈空时：说明当前 `')'` 没有匹配的左括号，把当前 `i` 压栈作为新的「无效分界」，后面算长度时不会越过它。
- ✅ 栈里始终是「未匹配的左括号下标」或「无效的右括号下标」，栈顶即「上一无效位置」。

### 写法二：动态规划

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        if n < 2:
            return 0
        dp = [0] * (n + 1)  # dp[i] 表示以 s[i-1] 结尾的最长有效括号长度
        for i in range(2, n + 1):
            if s[i - 1] == ")":
                if s[i - 2] == "(":
                    dp[i] = dp[i - 2] + 2
                else:
                    # 形如 "...( 有效段 )"，左括号在 j = i-1-dp[i-1]-1
                    j = i - 1 - dp[i - 1] - 1
                    if j >= 0 and s[j] == "(":
                        dp[i] = dp[i - 1] + 2 + dp[j]
        return max(dp)
```

- `dp[i]`：以 `s[i-1]` 结尾的最长有效括号长度。
- 只有当前字符为 `')'` 时可能变长：若前一个是 `'('` 则 `dp[i] = dp[i-2] + 2`；否则若前面有一段有效串，再前面是 `'('`，则把整段包起来并加上再前面的有效长度。

## 复杂度分析

| 写法 | 时间 | 空间 | 说明 |
|------|------|------|------|
| 栈   | O(n) | O(n) | 单遍遍历；栈最坏 n 个下标。 |
| DP   | O(n) | O(n) | 单遍填表；dp 数组长度 n+1。 |

---

## 易错点

- **栈初始为 `[-1]`**：若用空栈，第一个 `"()"` 弹栈后栈空，无法用 `i - stack[-1]` 得到长度 2。
- **弹栈后栈空**：必须 `stack.append(i)`，把当前 `')'` 的下标作为新的分界，否则后续会错误地跨过这个无效右括号。
- **边界**：空串返回 0；单字符返回 0；全 `'('` 或全 `')'` 返回 0。
