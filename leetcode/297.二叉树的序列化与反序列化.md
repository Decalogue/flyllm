# 297. 二叉树的序列化与反序列化

**标签**：`树` `设计` `困难`  
**难度**：⭐⭐⭐ 困难  

---

## 思路

前序 + 空结点标记（如 `None`）；反序列化用队列重建。

---

## 代码

```python
class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None


class Codec:
    def serialize(self, root):
        def dfs(n):
            if not n:
                return ["#"]
            return [str(n.val)] + dfs(n.left) + dfs(n.right)

        return ",".join(dfs(root))

    def deserialize(self, data: str):
        it = iter(data.split(","))

        def dfs():
            t = next(it)
            if t == "#":
                return None
            n = TreeNode(int(t))
            n.left = dfs()
            n.right = dfs()
            return n

        return dfs()
```
