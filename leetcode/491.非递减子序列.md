# 491. 非递减子序列

**标签**：`数组` `回溯` `去重` `中等`  
**分类**：DFS、子序列枚举  
**难度**：⭐⭐ 中等  
**频率**：🔥🔥 中频

---

## 题目描述

给你一个整数数组 `nums`，找出并返回所有该数组中不同的**非递减子序列**，并且每个子序列中**至少有两个元素**。

- 题目中的「递增」按力扣原题语义是**非递减**（允许相等），即后一个元素 `>=` 前一个元素。  
- 子序列不要求连续，但必须保持原相对顺序。  
- 结果中不能包含重复子序列。

### 示例 1

```
输入：nums = [4,6,7,7]
输出：[[4,6],[4,6,7],[4,6,7,7],[4,7],[4,7,7],[6,7],[6,7,7],[7,7]]
```

### 示例 2

```
输入：nums = [4,4,3,2,1]
输出：[[4,4]]
```

### 提示

- `1 <= nums.length <= 15`
- `-100 <= nums[i] <= 100`

---

## 📋 面试要点速查（知识卡片）

### 核心思路

**一句话总结**：DFS 枚举「选/不选」，路径保持**非递减**约束；每一层用一个 `used` 集合防止同层同值重复分支，路径长度 `>=2` 时收集答案。

**关键词**：`回溯` `非递减约束` `同层去重` `长度至少 2`

### 复杂度速记

- **时间复杂度**：最坏指数级，约 O(2^n) 到 O(n·2^n) 量级（输出规模主导）。  
- **空间复杂度**：O(n) 递归栈（不含答案存储）。

### 记忆口诀

「路径单调不下降；同层同值只走一条；长度到 2 就收集。」

---

## 解题思路

### 为什么是回溯

本题要的是**所有**合法子序列，不是最长长度（对比 [300. 最长递增子序列](300.最长递增子序列.md)）。最自然是 DFS 枚举子序列路径，并在搜索过程中做合法性与去重约束。

### 两个关键约束

1. **单调约束**（非递减）  
   - 若当前路径最后一个值为 `path[-1]`，则下一个候选 `nums[i]` 必须满足 `nums[i] >= path[-1]`。

2. **去重约束**（同层去重）  
   - 在同一递归层（固定 `start`）里，可能有多个相同值，比如 `[4,6,7,7]` 的两个 `7`。  
   - 若同层都尝试，会生成重复子序列。  
   - 做法：每层维护 `used = set()`，本层某个值处理过就跳过。

### DFS 框架

- 参数 `start`：下一次枚举起点（保证子序列顺序）。  
- `path`：当前已选子序列。  
- 若 `len(path) >= 2`，加入答案。  
- 从 `i = start` 往后枚举，按约束选择并递归 `dfs(i + 1)`，回溯撤销选择。

## 代码实现

### 写法一：标准回溯（推荐）

```python
class Solution:
    def findSubsequences(self, nums: list[int]) -> list[list[int]]:
        res: list[list[int]] = []
        path: list[int] = []

        def dfs(start: int) -> None:
            if len(path) >= 2:
                res.append(path[:])
            used: set[int] = set()  # 同层去重
            for i in range(start, len(nums)):
                if nums[i] in used:
                    continue
                if path and nums[i] < path[-1]:
                    continue
                used.add(nums[i])
                path.append(nums[i])
                dfs(i + 1)
                path.pop()

        dfs(0)
        return res
```

### 怎么读这段代码（写法一）

1. **`dfs(start)` 在枚举什么**  
   表示：接下来**只能**从 `nums[start:]` 里继续选数，拼到 `path` 后面。这样保证子序列在原数组里**下标递增**，不会乱序。

2. **先进 `if len(path) >= 2` 再进循环**  
   每进入一层 `dfs`，先看当前 `path` 是否已经够长；够长就**先拷贝一份**放进 `res`。注意：**空路径、单元素路径**也会走到这里，但只有 `len>=2` 才收集，所以不会出现长度 0、1 的答案。  
   同一 `path` 会在**后续继续选数**时多次作为前缀出现，因此会多次 `append` 更长的合法序列，这是预期行为。

3. **`used` 为什么写在 `dfs` 里面、每次递归新建**  
   `used` 只服务 **`for i in range(start, ...)` 这一层循环**：同一层里，**相同数值**只让**第一个还没被跳过的下标**去扩展，避免例如两个 `7` 在同一层各开一支、得到重复的 `[4,7]`。  
   递归进入下一层 `dfs(i+1)` 时会**重新** `used = set()`，所以**不同层**可以再次选同样的值（例如路径里需要两个 `7` 时，会在不同深度分别选）。

4. **两个 `continue` 分工**  
   - `nums[i] in used`：**同层去重**（见上）。  
   - `path and nums[i] < path[-1]`：**非递减约束**——只有当前路径非空时，新选的数不能比路径最后一个还小；`path` 为空时任意 `nums[i]` 都可作第一个数。

5. **四步一组：`used.add` → `append` → `dfs(i+1)` → `pop`**  
   标准回溯：选 `nums[i]`、往下搜、撤销选择。下一层从 **`i+1`** 开始，保证**不回头**用更靠前的下标，从而与「子序列」定义一致。

6. **和「子集型」回溯的差别（心里对照）**  
   这里不是简单的「选/不选当前下标」，而是「从 `start` 起挑一个位置接上」，并用 **`path[-1]`** 约束单调、用 **`used`** 约束同层重复值。

### 写法二：值域数组去重（利用 `nums[i]` 范围）

由于 `nums[i] ∈ [-100,100]`，可用长度 201 的布尔数组代替 `set`，常数略小：

```python
class Solution:
    def findSubsequences(self, nums: list[int]) -> list[list[int]]:
        ans: list[list[int]] = []
        path: list[int] = []

        def dfs(start: int) -> None:
            if len(path) >= 2:
                ans.append(path[:])
            used = [False] * 201
            for i in range(start, len(nums)):
                x = nums[i]
                if used[x + 100]:
                    continue
                if path and x < path[-1]:
                    continue
                used[x + 100] = True
                path.append(x)
                dfs(i + 1)
                path.pop()

        dfs(0)
        return ans
```

## ⚠️ 常见陷阱

1. **把「递增」写成严格递增**  
   - 本题允许相等，应是 `>=`，所以 `[7,7]` 合法。

2. **只做全局去重**  
   - 仅对最终结果 `set(tuple(path))` 去重虽可过，但效率差；推荐**同层去重**从源头减枝。

3. **把同层去重写成全路径去重**  
   - `used` 必须是每层局部变量；若放全局，会误删合法路径。

4. **和 300 题混淆**  
   - 300 求长度最优，通常 DP/二分；491 要枚举所有合法子序列，核心是回溯去重。

## 总结

491 的核心不是「最长」而是「**全部 + 去重**」：回溯维护非递减路径，按下标递进保证顺序，每层 `used` 去重避免重复分支。写出这一版基本就是面试标准答案。
