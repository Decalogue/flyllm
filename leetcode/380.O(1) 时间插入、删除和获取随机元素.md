# 380. O(1) 时间插入、删除和获取随机元素

**标签**：`哈希` `数组` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码

```python
import random


class RandomizedSet:
    def __init__(self):
        self.a = []
        self.pos = {}

    def insert(self, val: int) -> bool:
        if val in self.pos:
            return False
        self.pos[val] = len(self.a)
        self.a.append(val)
        return True

    def remove(self, val: int) -> bool:
        if val not in self.pos:
            return False
        i = self.pos.pop(val)
        last = self.a[-1]
        self.a[i] = last
        self.pos[last] = i
        self.a.pop()
        return True

    def getRandom(self) -> int:
        return random.choice(self.a)
```
