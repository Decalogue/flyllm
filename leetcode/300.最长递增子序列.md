# 300. 最长递增子序列

**标签**：`动态规划` `二分` `中等`  
**难度**：⭐⭐ 中等  

---

## 代码（耐心排序 / tails 二分）

```python
import bisect


class Solution:
    def lengthOfLIS(self, nums: list[int]) -> int:
        tails = []
        for x in nums:
            i = bisect.bisect_left(tails, x)
            if i == len(tails):
                tails.append(x)
            else:
                tails[i] = x
        return len(tails)
```
